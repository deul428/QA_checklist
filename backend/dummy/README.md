# Dummy 폴더

이 폴더에는 현재 사용하지 않는 스크립트와 문서 파일들이 보관되어 있습니다.

## 포함된 파일

### 데이터 임포트 스크립트
- `import_user_csv.py` - 사용자 CSV 데이터 임포트
- `import_csv_data.py` - 체크리스트 CSV 데이터 임포트
- `import_excel_data.py` - 엑셀 데이터 임포트
- `create_sample_data.py` - 샘플 데이터 생성

### 유틸리티 스크립트
- `delete_old_records.py` - 오래된 체크리스트 기록 삭제

### 테스트 스크립트
- `test_email.py` - 메일 발송 테스트
- `test_gmail_forwarding.py` - Gmail 포워딩 테스트
- `test_office365_smtp.py` - Office 365 SMTP 테스트
- `test_smtp_servers.py` - SMTP 서버 찾기 테스트

### 문서 파일
- `SMTP_SETUP_GUIDE.md` - SMTP 설정 가이드
- `FIND_SMTP_SERVER.md` - SMTP 서버 찾기 가이드
- `OFFICE365_SMTP_SETUP.md` - Office 365 SMTP 설정 가이드
- `user_passwords.json` - 사용자 비밀번호 정보 (보안 주의)

## 참고사항

- 이 파일들은 일회성 작업이나 테스트 목적으로 사용되었습니다
- 필요시 다시 사용할 수 있도록 보관되어 있습니다
- `user_passwords.json` 파일에는 민감한 정보가 포함되어 있을 수 있으므로 주의하세요
